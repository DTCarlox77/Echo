Registro de actualizaciones


Versión 1.6

Novedades:
  • Ahora es posible agregar espacios al nombre de una nueva sala.
Errores corregidos:
  • El texto con etiquetas HTML ya no será interpretado.
  • Se solucionó la falta de scroll en el área de texto para escribir tus mensajes.
  • Las cadenas largas de texto ya no causan desbordamiento.

Versión 1.5

Novedades:
  • EchoBot ahora tiene su propio diseño distintivo.
  • Los mensajes de los emisores ahora se alinean a la derecha.
  • Agregada barra de búsqueda de salas.
  • Seccionamiento de salas para evitar sobrecargar la página principal.
  • Posibilidad de enviar mensajes en formato Markdown (usando /md + 'mensaje').
  • Mejoras significativas de rendimiento.
Errores corregidos:
  • EchoBot corregido para evitar desconexiones por consultas inválidas, ahora responde con un mensaje de error.
  • Corregido el problema de desbordamiento con imágenes compartidas en la pantalla de mensajes.

Versión 1.4

Novedades:
  • Agregada la capacidad de redactar una biografía para tu usuario.
  • Mejoras en el diseño global de la aplicación.
  • Diseño responsivo para adaptarse a cualquier dispositivo.
  • Los creadores de salas ahora pueden expulsar usuarios (siempre y cuando la sala esté protegida por una contraseña de acceso).
Errores corregidos:
  • Buscar una sala ya no afectará al botón de carga de salas.
  • Corregido el problema de desbordamiento con archivos multimedia dentro de las salas.

Versión 1.3

Novedades:
  • Añadida función de búsqueda de salas (en las interfaces pública y personal).
  • Nueva página de perfil de usuario.
  • Accede al perfil de los usuarios tocando su nombre.
  • Visualización de la imagen de perfil de los usuarios en una sala.
Errores corregidos:
  • El mensaje 'Enviando archivo' ya no se quedará en la pantalla de forma permanente.

Versión 1.2

Novedades:
  • Visualización de un mensaje de progreso durante el envío de archivos.
  • Añadido botón para cargar salas fácilmente.
  • Nueva función para agregar una URL de imagen al perfil.

Versión 1.1

Novedades:
  • Añadidas nuevas páginas: Echo y Acerca de.
  • Introducida función de inteligencia artificial con el comando '/eb' + consulta.
  • Mejora en la interfaz: los mensajes ahora se ingresan en un área de texto.
  • En las salas se indica si un usuario está activo.
Errores corregidos:
  • Corregido el problema de envío de archivos con extensiones desconocidas.

Versión 1.0

Novedades:
  • Añadido sistema de registro e inicio de sesión.
  • Posibilidad de crear salas para compartir mensajes con amigos.
  • Nueva función de envío de archivos para compartir multimedia.
